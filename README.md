# She Can Foundation — Website

A simple, responsive website built for **She Can Foundation**, a non-governmental organisation registered under the Indian Societies Act, 1860, dedicated to empowering women and creating a more equitable society.

---

## Pages & Sections

- **Navbar** — Sticky navigation with smooth scroll links
- **Hero** — Full width banner with heading, tagline and call to action button
- **Carousel** — Auto-rotating image carousel with prev/next buttons and dot indicators
- **About** — Organisation description with image and text side by side
- **What We Do** — Four program cards covering Training, Advocacy, Community and Outreach
- **Join Us** — Call to action section with volunteer button
- **Footer** — Copyright and navigation links

---

## Technologies Used

- **HTML5** — Semantic structure and layout
- **CSS3** — Styling, layout and animations
- **JavaScript (Vanilla)** — Interactivity and DOM manipulation

---

## Features

- Responsive design that works on mobile and desktop
- CSS Flexbox and Grid for layout
- Sticky navbar
- Image carousel with auto-rotate every 3.5 seconds
- Clickable dot indicators synced to carousel
- Scroll fade-in animation on cards using IntersectionObserver API
- Smooth scrolling on all anchor links
- Volunteer button shows confirmation message on click
- Hover effects on cards, buttons and nav links
